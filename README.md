# カラータイル・ロマンス / Color Tiles Romance

色合わせパズル × 恋愛ノベルのブラウザゲーム。  
同じ色のタイルを直線で消しながら、5人の少女たちとの物語を進めていく。

**プレイ**: https://elnino033-creator.github.io/yukawayuami/

---

## ゲーム概要

| 項目 | 内容 |
|---|---|
| ジャンル | パズル × 恋愛ノベル |
| プラットフォーム | ブラウザ（PC / スマートフォン対応） |
| 技術スタック | TypeScript / Vite / Canvas API / Web Audio API |

### ストーリー

謎の招待状に導かれ「色彩の塔」に迷い込んだ主人公。  
塔に囚われた5人の少女（あかり・みお・すず・ひまり・ゆかり）を、色合わせのパズルを解きながら解放していく。

### キャラクター

| キャラ | 色 | 章 |
|---|---|---|
| あかり | ピンク | 第1章 緋色の階層 |
| みお | 青 | 第2章 氷雪の洗礼 |
| すず | 緑 | 第3章 翠葉の迷宮 |
| ひまり | オレンジ | 第4章 連鎖の宮殿 |
| ゆかり | 紫 | 第5章 混沌の頂点 |

### ゲームシステム

- **タイル消去**: 同色タイルを空きマス経由の直線で繋いで消す
- **ギミック**: 氷タイル・時間タイル・ブロック・連結タイル
- **スコア**: コンボ・タイムボーナス・ヒント未使用ボーナス
- **セーブ**: localStorage によるプレイデータ自動保存

## デバッグモード

URLに `?debug=1` を付けることでデバッグモードが有効になる。

**URL**: https://elnino033-creator.github.io/yukawayuami/?debug=1

### 起動方法

通常プレイURLの末尾に `?debug=1` を追加してアクセスする。

### 機能

画面左上に 🐛 ボタンが表示される。クリックするとデバッグパネルが開き、以下の操作が可能。

#### 📖 SCENARIOS（シナリオ一覧）

任意のシナリオを単体で再生できる。再生終了後はタイトル画面へ戻る（通常のゲームフローとは切り離されている）。

#### 🎮 STAGES（ステージ一覧）

任意のステージを直接起動できる。preScenario（ステージ前のシナリオ）はスキップされ、パズルが即座に開始される。

### ノベル画面の強制終了

デバッグモード中にシナリオを再生すると、コントロールバーに **→END** ボタンが表示される。クリックするとシナリオを即座に終了してタイトルへ戻る。

## 開発

```bash
cd color-tiles-romance
pnpm install
pnpm dev        # 開発サーバー起動
pnpm test       # テスト実行（71テスト）
pnpm typecheck  # 型チェック
pnpm build      # ビルド
```
