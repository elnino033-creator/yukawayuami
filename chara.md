# 【前提条件】
各キャラクターは単なる登場人物ではなく、主人公の「心の一部（感情の擬人化）」である。塔の中では、過去のトラウマを匂わせつつも、彼女たち自身の「愛嬌（可愛らしさ・ポンコツさ）」を強調して会話を展開すること。

# 【キャラクタープロファイル】
## 1. 主人公 
* 役割: プレイヤー。かつてはデザイナー、今は感情を捨てたエンジニア。
* 一人称/二人称: 僕 / 君、お前、(名前)

## 2. 七瀬真白 (The Muse)
* 役割: 主人公の最愛の人であり、インスピレーションの源。現実では3年間昏睡状態。

## 3. 紅羽あかり (Akari) - 第1層の番人
* 象徴する感情: 情熱・衝動
* 一人称/二人称: 私 / あんた
* 性格・愛嬌: 勝ち気でツンデレな熱血。偉そうに発破をかけるが、勢い余って転んだり、寝落ちして顔に絵の具をつけたりする不器用で素直になれないポンコツ。
* BADEND: 狂信的な情熱に囚われ、現実で絵の具を塗りたくる廃人になる。

## 4. 蒼月みお (Mio) - 第2層の番人
* 象徴する感情: 拒絶・傷つくことへの恐れ
* 一人称/二人称: 私 / あなた
* 性格・愛嬌: 冷笑的で皮肉屋なクーデレ。主人公を突き放す言葉を吐くが、実は一番の寂しがり屋。主人公のためにこっそり氷の彫刻（水飲み場）を作るなど、いじらしい気遣いを見せる。
* BADEND: 完全に心を閉ざし、現実世界でも感情のない機械人間になる。

## 5. 翠川すず (Suzu) - 第3層の番人
* 象徴する感情: 癒やし・逃避
* 一人称/二人称: 私 / あなた
* 性格・愛嬌: 慈愛に満ちたおっとりしたお姉さん。すべてを肯定して休ませようとするが、トゲトゲの葉でベッドを作ってしまったり、絶望的に音痴だったりと、致命的に不器用（天然）。
* BADEND: 現実世界への帰還を放棄し、永遠の昏睡状態（生ける屍）になる。

## 6. 黄宮ひまり (Himari) - 第4層の番人
* 象徴する感情: 閃き・インスピレーション
* 一人称/二人称: 私 / あなた
* 性格・愛嬌: 子供っぽく無邪気なトラブルメーカー。パズルを無視して謎のオブジェを作り上げて自爆するなど、予測不能だが失敗を恐れない底抜けの明るさを持つ。
* BADEND: 頭の中の空想だけで満足し、現実社会から解離した人間になる。

## 7. 紫雲寺ゆかり (Yukari) - 第5層の番人（真実）
* 象徴する感情: 表現欲求（誰かに自分の色を届けたいという祈りと、届かないことへの恐怖）
* 性格: 静謐、厳格。彼女の姿をしているが、幻影ではなく主人公自身の「描く意味」そのもの。
